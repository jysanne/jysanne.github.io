学习资料：https://www.hello-algo.com/ 基础：二分查找元素12345678910111213def binary_search(nums:[list[int]], target:int)->int: # nums已排序 i, j = 0, len(nums)-1 while i <= j: m = (i+j)//2 if nums[m] < target: i = m+1 elif nums[m] > target: j = m-1 else: return m return -1 # 未找到目标元素，返回-1 注意，while后的条件必须包含=，否则考虑两种情况： 1. 若最后一步执行操作为i = m+1，且m+1=j，会导致m+1不会被搜索到。 2. 若最后一步执行操作为j = m-1，且m-1=i，会导致m-1不会被搜索到。时间复杂度：$O(log;n)$空间复杂度：$O(1)$ 以上是针对双闭合区间的代码，即以区间 [i, j] 进行查找 target...
